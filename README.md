# Group2US
Taller colaborativo de git

## Uso

Desde la carpeta del proyecto puede ejecutar el menú interactivo:

```powershell
python d:\MATERIAS INGENIERIA\segundo cuatrimestre\programación\Factoriales\Group2US\main.py
```

Para calcular un factorial directamente desde Python:

```python
from factoriales import factorial
print(factorial(5))  # 120
```

Tests: Si desea ejecutar los tests (requiere pytest instalado):

```powershell
python -m pip install pytest
python -m pytest -q
```
